Documentation

Présentation

L'objectif du projet est de réaliser un site web en se basant sur un site web des masters de l'université des Aix-Marseille déjà existant.

le Travail a faire:

  1. Réaliser une base de données sous format XML , qui décrit le mieux possible l'ensemble des infos sur les intervenants, les parcours et l'ensemble des unités d'enseignements.

    il faut prévoir une DTD et une schéma xsd qui permettront de valider le contenue du fichier Xml

    La base de données du site existant a été fournie, donc il faut prévoir une transformation vers une structure valide pour ce projet

  2. Réaliser un XSLT (version 2.0) affin de générer les pages du nouveau site.

  3. Réaliser une requête XQuery pour extraire toutes les infos sur les parcours en responsabilité et les unité d'enseignement assurées pour tout les intervenants.

  4. Réaliser un code java qui permet de extraire les nom des unités d'enseignements et retourner le résultat sur la sortie standard sous le format d'un fichier XML

  5. un Makefile pour le projet

Structure du projet

  • masterf.xml : base de données sous format XML
  • dtd/master.dtd : dtd qui permet de valider le fichier masterf.xml
  • xsd/master.xsd : schéma XSD qui permet de valider le fichier masterf.xml
  • xsl/master.xsl : XSLT (version 2.0) qui génère les pages du site
  • css/ : feuilles de style du site (Bootstrap)
  • img/ : images du site (un logo de l'université des Aix-Marseille)
  • xq.txt : requête XQuery pour extraire toutes les infos sur les parcours en responsabilité et les unité d'enseignement assurées pour tout les intervenants.
  • java/MakeDom.java : code java qui permet de extraire les nom des unités d'enseignements et retourner le résultat sur la sortie standard sous le format d'un fichier XML.
  • saxon9pe.java : outil saxon
  • Makefile : fichier qui offres plusieurs options (voir partie utilisation)
  • documentation.html : cette page :D

Utilisation

Warning! vérifier que packet tidy est installé sur votre machine

Avec le Makefile, il est possible de générer le site web plus un ensemble taches.

vous devez dabord ouvrir le terminal et vous placez sur le répertoire racine du projet ou se trouve le fichier Makefile.

Maintenant vous pouvez exécuter la commande make all et attendre que le site s'affiche sur votre navigateur préféré :D

Si vous êtes si curieux voici toutes les Commandes Disponible:

  • make all: exécute les tache sur cette ordre => reset dtd xsd javap web xq tidy open
  • make web: Création du contenu du dossier www/ (les page web pour chacun des intervenants, des parcours et l'ensemble des unités d'enseignements )
  • make dtd: Validation du masterf.xml avec la DTD
  • make xsd: Validation du masterf.xml avec la schéma
  • make xq: Création du résultat de la requête XQuery => création du fichier www/xq.html
  • make tidy: Validation des fichiers www./*.html
  • make javap: Création du fichier listues.xml contenant l'ensemble des UEs
  • make reset: Nettoyer le projet en exécutant l'ensemble des commandes suivantes:
    • rm -rf www/*
    • rm -rf www
    • rm -rf listues.xml
    • rm -rf java/*.class
  • make open: Afficher le site sur votre Navigateur par défaut
